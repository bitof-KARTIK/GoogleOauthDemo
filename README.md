
## Setup

1. **Clone the repo and install dependencies:**

   git clone <repository-url>
   cd GoogleAuthExpress
   npm install
   

2. **Create a `.env` file:**
   

3. **Install Dependencies**

   npm i

4. **Run The App**

   npx nodemon app.js
   
   

5. **Visit `http://localhost:3000/auth/google` to start authentication.**
